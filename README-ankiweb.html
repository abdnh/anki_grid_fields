Allows you to display the editor in multiple columns, or any arrangement you want. It knows three modes:
  1. Auto Columns: It will generate the appropriate amount of columns for you, depending on the width of the editor window. It is active, when the column number is set to 0.
  2. Grid Columns: You can also manually set the amount of columns per hand (in the editor toolbar). This is the behavior, when the column number is set to 1 - 9
  3. Grid Areas: Advanced feature. You can define how the fields are displayed yourself. You can set this behavior under "Editor > Fields > Grid". If you enable it, it will overwrite the Auto columns as the behavior when the column number is set to 0. See below for more details.
Auto Columns in a smaller window, and in a bigger window. Notice the "Cols" toggle in the toolbar for setting the column count by hand: Grid areas: This arrangement would have the definition: f1 f1 f2 f3 f4 f5 f6 f7 f5 This definition is what you have to enter into the dialog under "Editor > Fields > Grid". Shortcuts
  • F12: Toggle between Auto columns and Grid areas in the editor.
  • Shift+F12: Make a field temporarily full width (works well with Auto/Grid Columns, will obscure fields when using Grid areas).
