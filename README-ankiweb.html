Allows you to display the editor in multiple columns, or any arrangement you want. It knows two modes:
  1. Auto Columns: It will generate columns for you, which automatically adjust to the width of the editor window. This is the default behavior.
  2. Grid areas: You can define the areas how the fields are displayed yourself. You can set this behavior under "Editor > Fields > Grid". See below for more details.
Auto columns in a smaller window, and in a bigger window: Grid areas: This arrangement would have the definition: f1 f1 f2 f3 f4 f5 f6 f7 f5 This definition is what you have to enter into the dialog under "Editor > Fields > Grid".
