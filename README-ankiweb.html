Allows you to display the editor in multiple columns, or any arrangement you want. It knows two modes:
  1. Grid Columns: It will generate columns for you. If you set the column number to 0, it will automatically adjust to the width of the editor window. You can also adjust the amount of columns per hand (in the editor toolbar). This is the default behavior.
  2. Grid areas: You can define the areas how the fields are displayed yourself. You can set this behavior under "Editor > Fields > Grid". See below for more details.
Columns in a smaller window, and in a bigger window. Notice the "Cols" toggle in the toolbar for setting the column count by hand: Grid areas: This arrangement would have the definition: f1 f1 f2 f3 f4 f5 f6 f7 f5 This definition is what you have to enter into the dialog under "Editor > Fields > Grid". You can use F12 to toggle between Grid columns and Grid areas in the editor. You can use Shift+F12 to make a field temporarily full width (works with Grid Columns, not always with Grid areas).
